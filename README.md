# 📚 Smart Library Management System

A **Spring Boot + PostgreSQL** based web application designed to efficiently manage library operations such as adding, updating, deleting, and searching books.

---

## 🎯 Project Objective
To develop a digital library system that automates manual book management processes and provides fast, reliable, and structured access to library data.

---

## 🚀 Features
- Add new books to the system  
- Update existing book details  
- Delete books from database  
- Search books by title, author, or category  
- View all available books  
- RESTful API integration  

---

## 🛠️ Tech Stack
- Java  
- Spring Boot  
- Spring Data JPA  
- PostgreSQL  
- REST API  
- Maven  

---
## ⚙️ Setup Instructions
1. Clone the repository  
2. Create PostgreSQL database:

CREATE DATABASE librarydb;

3. Update `application.properties` with your database credentials  
4. Run the Spring Boot application  
5. Access APIs at:

http://localhost:8080
